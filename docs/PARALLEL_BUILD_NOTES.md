# Parallel Build Notes

This repo may be developed in parallel with the existing Codex/current repo.

The goal is not a competition and not a porting exercise. The goal is to let two independent builders explore the same product mission from different starting points, then compare working outcomes.

## Independence Rule

Claude should not inspect or copy the sibling Codex/current repo unless the user explicitly asks.

Use this repo's own startup docs:

- `docs/PRODUCT.md`
- `CLAUDE.md`
- `docs/CLAUDE_BUILD_BRIEF.md`
- `docs/CLAUDE_WORK_LOOP.md`
- `docs/research/README.md`

Codex reference docs under `docs/reference/` are opt-in context only.

## Later Comparison

Compare builds by behavior, not by who followed whose plan:

- Track Master UI feel.
- Real-time audition quality.
- Original/Mastered A-B reliability.
- Export speed and trustworthiness.
- Audio quality on private fixtures.
- Album Master readiness.
- Architecture clarity.
- Test and verification evidence.

Do not merge ideas across repos until the user asks for comparison or consolidation.
